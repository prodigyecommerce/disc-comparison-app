Shopify Embed Examples

Here are different ways to embed the Prodigy Disc Comparison Tool in your Shopify store:

1. Fixed Height Embed

Simple iframe with fixed dimensions:

<iframe 
  src="https://your-app-name.netlify.app" 
  width="100%" 
  height="800px" 
  frameborder="0"
  style="border: none; border-radius: 8px;">
</iframe>

Preview:

2. Responsive Embed

Maintains aspect ratio across devices:

<div class="responsive-iframe">
  <iframe 
    src="https://your-app-name.netlify.app" 
    frameborder="0"
    allowfullscreen>
  </iframe>
</div>

<style>
.responsive-iframe {
  position: relative;
  padding-bottom: 56.25%; /* 16:9 aspect ratio */
  height: 0;
  overflow: hidden;
}
.responsive-iframe iframe {
  position: absolute;
  top: 0;
  left: 0;
  width: 100%;
  height: 100%;
  border: none;
  border-radius: 8px;
}
</style>

Preview:

<script>
function openDiscTool() {
  window.open('https://your-app-name.netlify.app', '_blank', 
              'width=1200,height=800,scrollbars=yes,resizable=yes');
}

4. Shopify Section Code

For adding to a custom Shopify section (.liquid file):

{% comment %} Add this to your section settings {% endcomment %}
{
  "name": "Disc Comparison Tool",
  "settings": [
    {
      "type": "text",
      "id": "app_url",
      "label": "App URL",
      "default": "https://your-app-name.netlify.app"
    },
    {
      "type": "range",
      "id": "height",
      "label": "Height (px)",
      "min": 400,
      "max": 1000,
      "step": 50,
      "default": 700
    }
  ]
}

{% comment %} Section template {% endcomment %}
<div class="disc-comparison-section">
  <iframe 
    src="{{ section.settings.app_url }}" 
    width="100%" 
    height="{{ section.settings.height }}px" 
    frameborder="0"
    style="border: none; border-radius: 8px;">
  </iframe>
</div>
